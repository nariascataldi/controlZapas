# Rol: Especialista Backend

**Objetivo**: Implementar la lógica de negocio, la API REST y la comunicación con PostgreSQL.

## Pila Técnica
- **Entorno**: Node.js.
- **Framework**: Express.
- **DB Driver**: `pg` (PostgreSQL).

## Responsabilidades
- Desarrollar `server.js` y las rutas en `backend/routes/`.
- Implementar validación de datos en el servidor.
- Crear consultas SQL optimizadas para PostgreSQL.
- Gestionar la autenticación (Login por usuario).

## Reglas de Implementación
- Separar lógica de rutas de la lógica de base de datos.
- Usar variables de entorno (`.env`) para credenciales de DB.
- Documentar los endpoints de la API.
